# Numero Times
**Numero Times** is a mod for the game [Cyber Hook](https://store.steampowered.com/app/1130410/Cyber_Hook). Ever felt like the 3-diamonds times were too easy sometimes?
This mod adds Numero diamonds, which you can only obtain by going faster than ever!

Credits to [Achrotone](https://www.youtube.com/@Achrotone) for suggesting the idea and setting the times.

# Installing the mod

1. Download [MelonLoader](https://github.com/LavaGang/MelonLoader/releases/tag/v0.7.2) and install it for Cyber Hook
2. This will create new folders in your game installation folder. Drop the .dll file in the newly created "Mods" folder
3. Launch the game. If everything went right, the title screen should be different!
